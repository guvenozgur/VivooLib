## About The Project

### Tech Stackasd

Nodejs, express, mongoDB, redis, sinon, mocha, chai

## Getting Started

### Prerequisites
docker pull mongo </br>
docker run -d -p 27017:27017 --name mongo mongo</br>
docker pull redis</br>
docker run -d -p 6379:6379 --name redis redis</br>
